Adds the Digzign's button to your WordPress posts and/or pages.

== Description ==

This plugin installs the Digzign Button on your Posts (blog and/or individual posts) and/or pages. It supports all the <a href='http://digzign.com/tools/button'>settings available</a>.

== Known Bugs ==
Unknown, at the moment.

== Installation ==

1. Unzip/upload the plugin to your wp-content/plugins.
2. Enable it on your plugins control panel.
